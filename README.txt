# Komorebi Cafe LP

古民家カフェ向けの集客用ランディングページ（LP）です。

AIを活用しながら、Next.js・TypeScript・Tailwind CSSで制作したポートフォリオ作品です。

## Demo

https://komorebi-cafe-lp.vercel.app/

## Overview

「築100年の古民家で過ごす、静かなごほうび時間。」をコンセプトにした架空の古民家カフェのLPです。

スマートフォン閲覧を重視し、予約導線・メニュー紹介・店舗情報・FAQを分かりやすく配置しています。

## Features

* モバイルファースト設計
* レスポンシブ対応
* ヒーローセクション
* 人気メニュー紹介
* ギャラリー
* お客様の声
* FAQアコーディオン
* Google Maps埋め込み
* Instagram導線
* 電話予約導線

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Vercel
